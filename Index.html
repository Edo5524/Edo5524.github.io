Tujuan Pembuatan Website Portfolio

Website portfolio ini dibuat dengan tujuan untuk memperkenalkan diri secara profesional serta menampilkan hasil karya dan proyek yang telah saya kerjakan. Beberapa tujuan utama dari website ini adalah:

  • Menampilkan informasi pribadi dan profil secara profesional.
  • Memperlihatkan daftar program atau project yang pernah saya buat.
  • Sebagai sarana untuk mempermudah orang lain menghubungi saya melalui halaman kontak.
  • Menjadi portofolio online yang dapat diakses kapan saja untuk kebutuhan pekerjaan atau klien.
